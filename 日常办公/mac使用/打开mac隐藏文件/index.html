打开mac隐藏文件

发表于2026-04-11|更新于2026-04-23|日常办公mac使用

你的 Mac 里藏着一个神秘世界。本文教你用最简单的方式找到并打开那些「看不见」的文件夹。

1. 什么是隐藏文件夹？

在 macOS 中，以 .（英文句点）开头的文件夹默认是隐藏的，比如：

  • ~/.ssh — SSH 密钥目录
  • ~/.bash_profile — 命令行配置文件
  • ~/Library — 应用支持数据目录
  • .Trash — 废纸篓

Finder 一开始不会显示它们，但它们确实存在。有时候你需要修改配置、清理缓存、查看日志，都得先找到它们。

2. 方法一：让 Finder 显示所有隐藏文件

如果你想彻底让 Finder 显示所有 . 开头的文件和文件夹，用这条命令：

  1. 打开Finder
  2. 按快捷键 Command + Shift + .，显示隐藏文件夹

⚠️ 注意：显示隐藏文件后，桌面上会出现一些系统配置文件（如 .DS_Store），视觉上会有点乱，用完记得关掉。

Step 1：打开 Terminal

按 Command + Space 调出 Spotlight，输入 Terminal，回车。

4. 方法三：快捷键「前往文件夹」（最简单）

如果你的目标路径已知，这个方法最快：

  1. 打开 Finder
  2. 按快捷键 Command + Shift + G

这个方法不需要显示隐藏文件，就能直接打开任意文件夹，包括隐藏的。

查看 SSH 密钥 open ~/.ssh

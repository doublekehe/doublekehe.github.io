markdown文章中插入图片

发表于2026-03-01|更新于2026-03-21|代码编程markdown

Markdown 中插入图片的语法如下：

  • ！：感叹号 ! 是必须的，没有它会被解析为普通链接。
  • […]：方括号内是替代文本（alt text），当图片无法加载时显示该文本，也用于视障用户访问。
    • 类似这种
  • (…)：圆括号内是图片的路径（URL 或本地路径）。
  • “…”（可选）：双引号内是鼠标悬停时显示的图片标题（title）。

图片路径可以用本地文件路径，也可以使用网络图片url。本地相对路径（最常用），适用于本地 Markdown 文件（如笔记、博客源文件）。推荐使用相对路径，避免使用绝对路径，否则在跨设备或者git仓库中会失效。

网络图片（URL），直接引用在线图片，依赖网络，可能被防火墙拦截或链接失效

如果需要调整图片尺寸和位置，可以试用类似html的语言，比如

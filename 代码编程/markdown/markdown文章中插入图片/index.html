1. Markdown 图片语法

Markdown 插入图片最常用的是下面这个格式：

  • !：感叹号是必须的。没有它，Markdown 会把内容解析成普通链接。
  • []：方括号里写替代文本，也就是 alt text。图片加载失败时会显示这段文字，读屏软件也会读取它。
  • ()：圆括号里写图片路径，可以是本地相对路径，也可以是网络图片地址。
  • "可选标题"：鼠标悬停在图片上时显示的标题，可以不写。

常用样式可以记住这几个：

样式 作用

max-width:100% 防止图片在小屏幕上溢出

text-align:center 让内部图片居中显示
object-fit:cover 固定宽高时裁剪图片，适合封面图

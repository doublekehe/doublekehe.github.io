git本地文件同步到新仓库

发表于2026-03-07|更新于2026-03-07|代码编程git

本地有一些重要的文件，想要同步到git上怎么操作呢？

1. 在 GitHub 创建空仓库

  • 登录 GitHub → 右上角 + → New repository
  • 填写仓库名（如 my-project）
  • 关键设置：
    • ❌ 不要勾选 “Add a README file”
    • ❌ 不要勾选 “.gitignore” 或 "License"💡 原因：避免远程仓库有文件导致本地推送冲突
  • 点击 Create repository

2. 进入本地文件夹并初始化git仓库

成功后会生成隐藏的 .git 文件夹

4. 关联远程仓库并推送代码

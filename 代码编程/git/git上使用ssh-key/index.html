git上使用ssh key

发表于2026-03-13|更新于2026-03-13|代码编程git

在git中使用SSH密钥是为了安全地连接到远程仓库，而不需要在每次推送或拉取时输入用户名和密码。

1. 查看是否已有ssh key

如果看到以下文件之一，说明你已有密钥：

  • id_rsa 和 id_rsa.pub（传统 RSA）
  • id_ed25519 和 id_ed25519.pub（更安全的 Ed25519，推荐使用）

2. 生成密钥

参数说明：

  • -t rsa：指定密钥的类型为RSA
  • -b 4096：指定密钥的位长度为4096位（这是一个安全的长度）
  • -C：后面是你的电子邮件地址，用于标识密钥

默认会在目录 ~/.ssh/ 中生成两个文件：id_rsa 和 id_rsa.pub。

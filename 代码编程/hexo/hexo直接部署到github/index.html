hexo直接部署到github

发表于2026-03-03|更新于2026-03-21|代码编程hexo

Hexo 框架中将生成的静态网站文件推送到远程服务器或代码托管平台（如 GitHub Pages、Gitee Pages 等）的过程称之为部署，常用hexo deploy（简写hexo d）命令完成

安装部署插件

Hexo 默认不包含部署功能，需安装对应插件来部署到 Git 平台（GitHub / Gitee / GitLab）

配置 _config.yml

在博客根目录的 _config.yml 底部添加部署信息：

关键字段说明：

  • type: 固定为 git（使用 hexo-deployer-git 插件）；
  • repo: 仓库的 HTTPS 或 SSH 地址（推荐 SSH 免密）；
  • branch: 目标分支（GitHub 新仓库默认 main，旧仓库可能为 master）。

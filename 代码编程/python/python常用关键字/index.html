python常用关键字

发表于2026-04-22|更新于2026-04-23|代码编程python

Python 中的关键字是python语言内置的特殊标识符，不能用作变量名、函数名等标识。比如我们常用的if、while、True、and等等。

可以用过keyword类来查看当前python版本下对应的关键字（在该版本下，一共是35个关键字）

类型 关键字
流程控制 if — 条件判断
elif — 否则如果
else — 否则
for — 循环
while — 循环
break — 跳出循环
continue — 跳过本次循环
pass — 空操作（占位符）
逻辑运算 and — 逻辑与
or — 逻辑或
not — 逻辑非
True — 布尔真
False — 布尔假
None — 空值（类似其他语言的 null）
函数与类定义 def — 定义函数
class — 定义类
return — 返回值
yield — 生成器返回值
lambda — 匿名函数
异常处理 try — 尝试执行
except — 捕获异常
finally — 无论是否异常都执行
raise — 抛出异常
assert — 断言
模块与导入 import — 导入模块
from — 从…导入
as — 别名
变量作用域 global — 声明全局变量
nonlocal — 声明非局部变量（闭包中使用）

上下文管理 with — 上下文管理器

case — 匹配分支
其它关键字 del — 删除对象
in — 成员检查（也用于 for 循环）

python常用操作符

发表于2026-03-02|更新于2026-03-02|代码编程python

Python 提供了丰富的操作符，分为以下几大类：

  • 算术操作符：+、-、、/、//、%、*
  • 比较操作符：==、!=、<、>、<=、>=、
  • 赋值操作符：=、+=、-=、/=、*=、**=
  • 逻辑操作符：and、or
  • 位操作符：&、^、|、~、<<、>>
  • 成员操作符：in、not in

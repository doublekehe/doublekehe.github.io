Python 提供了丰富的操作符，分为以下几大类：

  • 算术操作符：+、-、*、/、//、%、**
  • 比较操作符：==、!=、<、>、<=、>=、

  • 逻辑操作符：and、or
  • 位操作符：&、^、|、~、<<、>>

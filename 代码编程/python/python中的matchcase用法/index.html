python中的matchcase用法

发表于2026-04-23|更新于2026-04-23|代码编程python

Python 3.10 引入了 match-case 语句，这是 Python 语言的一个重要更新，提供了强大的模式匹配功能。类似于其他语言中的 switch-case，但功能更强大，可以根据数据的“模式”或“结构”来执行不同的代码块，匹配更复杂的模式。

基本用法：值匹配

在最简单的场景下，match-case 可以用来替代冗长的 if-elif-else 链，使代码更清晰。

要点：

  • match 后面跟着一个表达式（如变量 status_code）。
  • case 后面跟着一个模式（如字面值 200）。
  • 程序会从上到下依次尝试匹配，一旦找到匹配项，就执行对应的代码块，然后自动退出 match 结构，不需要 break 语句。
  • case _ 是通配符模式，可以匹配任何值，通常作为默认情况放在最后。

进阶用法：结构化模式匹配

match-case 还可以直接匹配列表或元组的结构和内容，以及使用 | (或) 运算符可以在一个 case 中匹配多个值，还可以在case后增加if条件做筛选

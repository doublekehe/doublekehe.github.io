python常用关键字yield

发表于2026-04-24|更新于2026-05-09|代码编程python关键字

yield是Python中用于定义生成器(Generator)的关键字，主要区别于return关键字，yield可以将普通函数转换为生成器函数，从而能够按需生成值，而不是一次性计算所有值。

与普通函数使用 return 返回一个值并结束不同，yield 可以在函数执行过程中暂停并返回一个值，下次调用时从上次暂停的地方继续执行。

让函数有如下特性：

  • 惰性求值：只有当使用 next() 函数或者 for 循环遍历这个生成器对象时，函数才会开始执行。
  • 状态保持：函数执行到 yield 语句时，会暂停并返回一个值。此时，函数的所有局部变量和执行位置都会被“冻结”保存起来。当下次调用 next() 时，函数会从上次暂停的地方继续执行，而不是从头开始 。

yield和return的区别：

  • 普通函数：调用一次，执行完就结束，返回一个结果。
  • 生成器函数：调用后返回一个 生成器对象，可以多次“产出”（yield）值，状态被保存。

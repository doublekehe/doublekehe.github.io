python关键字lambda

发表于2026-04-24|更新于2026-04-24|代码编程python

lambda 是 Python 中用于创建匿名函数的关键字。可以定义一个简单、单行的函数，而无需使用 def 语句命名。lambda 不是为了替代 def，而是为了在需要短小函数对象时提供便利。

  • 只能包含一个表达式（不能是多条语句或复杂逻辑）
  • 自动返回表达式的值
  • 没有函数名

lambda 返回的是函数对象，可以赋值给变量，也可以直接调用。

lambda与高阶函数配合使用（如 map, filter, sorted）

  • map 示例：对列表每个元素平方
  • filter 示例：筛选偶数
  • sorted 示例：按字典的某个键排序

python中的map函数

发表于2026-04-29|更新于2026-05-09|代码编程python常用函数

map() 是 Python 内置函数，作用是对可迭代对象的每个元素执行同一个操作。

基本语法

最简单的例子：每个数字乘以 2

注意：map() 返回的是一个迭代器，要用 list() 转一下才能看到结果。

常见用法

1. 配合内置函数

2. 配合 lambda

3. 配合自定义函数

4. 多个可迭代对象（同时处理两个列表）

map vs 列表推导式：

两者效果一样，这两行等价

一般来说：

  • 用现成函数（如 int、str、len）→ map 更简洁

  • 逻辑复杂 → 列表推导式更直观
